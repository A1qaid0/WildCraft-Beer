# WildCraft Beer

Statyczna strona internetowa browaru WildCraft Beer.

## Zawartość

- stała oferta oraz limitowane serie sezonowe,
- historia browaru, wartości marki oraz opis procesu warzenia,
- interaktywne opisy pięciu stylów piwa i animowane liczby procesu,
- propozycje food pairingu oraz opinie klientów,
- interaktywny Beer Finder dobierający piwo do preferowanego smaku,
- sekcja dla różnych grup odbiorców oraz mini blog,
- galeria z lightboxem, formularze kontaktowe i interaktywna mapa.

## Uruchomienie

Otwórz plik `index.html` w przeglądarce lub umieść cały katalog na hostingu statycznym.

## Ważne informacje

- Interaktywna mapa korzysta z OpenStreetMap i wymaga połączenia z internetem.
- Formularz kontaktowy oraz zapis do newslettera otwierają domyślny program pocztowy użytkownika.
- Katalog `assets` musi pozostać obok pliku `index.html`.
